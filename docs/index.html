sevenrad-stills

Poetic interpretation of digital media through algorithmic transformation

Features

Compression & Degradation

JPEG compression, multi-generation artifacts, downscaling, and motion blur operations

Degradr Effects

VHS artifacts, sensor noise, chromatic aberration, Bayer filters, and bokeh effects

Non-Destructive Workflow

Incremental versioning preserves every transformation step with deterministic operations

YAML Pipelines

Declarative configuration system for complex multi-step transformation workflows

Quick Start

Install with mise (manages Python and ffmpeg):

mise install
uv venv
source .venv/bin/activate
uv pip install -e ".[dev]"

Run your first pipeline:

sevenrad pipeline docs/tutorials/compression-filters/01-social-media.yaml
